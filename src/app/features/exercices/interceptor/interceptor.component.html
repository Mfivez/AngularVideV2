Exercices - Interceptor

Enoncé :

Utiliser un Interceptor pour ajouter un token d'authentification et gérer les erreurs.

Objectifs :

Créer un interceptor qui ajoutera un token d'authentification (simulé) dans chaque requête sortante

Gérer les erreurs globales, par exemple, si une requête retourne un stat 401 (Unauthorized), rediriger l'utilisateur vers la page de login.

(Optionnel) : Ajouter un loader qui s'affiche pendant les requêtes HTTP.
